tryLoginPopup

  • tryLoginPopup(): Promise<"SUCCESS" | "ALREADY_LOGGED_IN" | "POPUP_FAILED">
    • Defined in index.ts:377

    Tries to open a popup to login the user in. If the user is already loggedIn() it will immediately return 'ALREADY_LOGGED_IN'. If the popup fails to open, it will immediately return 'POPUP_FAILED' else it will wait for loggedIn() to be true and eventually return 'SUCCESS'.

    Also dispatches login event

    Returns Promise<"SUCCESS" | "ALREADY_LOGGED_IN" | "POPUP_FAILED">
